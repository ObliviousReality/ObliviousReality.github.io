Tom's Epic Chess was the first game I ever made. It was an assignment for my A-Level computer science project, with no real design specifications other than it had to be built in Python. At the time I didn't know how to implement Object Oriented Programming into Python (or any language for that matter), so the whole project is procedural, and 99% of it is done in one file. It's 1200 lines long, vaguely commented and I have no idea how it actually works. Surprisingly, it does actually work, so I don't question it, don't touch it and I definitely don't look at it for too long - You never know, there may be a crazed murderer on line 947. He may mistake you for his target as no one bothered to tell him what the real target looked like. Bring a brace, you may be able to scare it off.

The brief of the project was simple: make something in Python. How this escalated into perhaps the best/worst game of chess ever made is lost to time. It's made using Python's Pygame library, a fairly powerful library for creating 2D graphics in a window, and one I quite like as it has extensive documentation on pygame.org.

The graphics, which for their time were awful, are a clear insight into my mind - complete utter laziness. The only 'quality' graphics are the chess pieces, which, unsurprisingly, are the only bits I didn't make - they were kindly donated by Oliver Hewison, who was pretending to be my 'professional' client for this task. Most of the UI text is yellow because I accidentally set that as the default colour for text and then never changed it.


The earlier versions of the program had music, originally the Kahoot theme, as suggested by Oli, but I later changed it to the Skyrim Theme, to better complement the gory pawn on pawn 0 frame unanimated combat the game has. This was then removed from the final version to 'clean' the program up a bit, along with the original end screen image, which was from one of the very few game worse than Tom's Epic Chess, Big Rigs: Over the Road Racing.


So that's Tom's Epic Chess. Never coming to a store near you.
